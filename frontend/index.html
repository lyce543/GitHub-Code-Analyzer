How it works

Three steps to full understanding

01
Paste a GitHub URL

Drop any public repository link into the input field and hit Analyze. CODEX clones it instantly via git and prepares the files for indexing.

02
AI Indexes the Code

LangChain splits all Python files into chunks, creates vector embeddings with OpenAI, and stores them in FAISS — all locally on your machine.

03
Ask Anything, Get Answers

Type your question in the chat. CODEX retrieves the most relevant code chunks, feeds them to GPT-4o, and streams back a precise answer in real time.
